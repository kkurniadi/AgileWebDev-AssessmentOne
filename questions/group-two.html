Content Questions (Group Two)

What are programming controls and design structures?

Programming controls are a way of specifying the flow of control in a program.
This can be achieved using design structures, which can be described as basic building blocks of logic.
These structures include Sequence, Selection and Iteration.

What are website testing procedures?

Website testing procedures are the steps that ensure that an application is fully functional and runs smoothly and securely.
They are an essential part of web development and ensure that an app is running properly before it is released.
There are various things that must be tested, such as functions, usability, interactions, compatibility, performance and security.

What are website coding techniques?

Website coding techniques are the ways in which a website can be coded.
It is possible to code a website through use of HTML and CSS.
Alternatively, a framework such as Bootstrap can be implemented.
Finally, there are services such as WordPress that allow people to build websites without them having to write lines of code.
