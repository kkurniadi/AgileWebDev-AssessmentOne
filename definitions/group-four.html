Content Definitions (Group Four)

Provide a definition for the following types of technologies.

Sequence
A sequential ordering of execution for the blocks of instructions
Selection
Algorithm selecting one of the alternate paths to follow
Repetition
A set/block of instructions that is executed zero, one or more times

HTML is the language in which most websites are written. It is used to create pages and make them functional.
Hypertext means that the document contains links to other sections of the document, or to other documents altogether.
Markup language is how computers speak to each other and control how text is processed or presented.
In HTML, this is done using tags and attributes.

CSS is the language used to style the visual appearance of web pages.
It tells web browsers how to render the different parts of a web page.
Background images, colours, borders, font size, typography and position of items on a web page are all defined by CSS.

Syntax is the rules that define the structure of a language.
In computer programming, this refers to the rules that control how the symbols, punctuation and words of a programming language can be combined in order to create statements or expressions in that language.
Without syntax, it would be nearly impossible to understand how a programming language works.
